# DICOM Sync

A cross-platform DICOM synchronization and query tool built using **Avalonia UI** and **.NET**.

### Features
- Dual PACS comparison (Left PACS vs. Right PACS)
- AE Management (AE Title, Hostname, Port, Echo)
- Query controls with global date range selector
- TreeView for browsing study/series data
- Sync actions:
  - Query Left
  - Query Right
  - Left → Right
  - Right ← Left
  - Sync
  - Pause

### Technology
- Xojo
- dcmtk
